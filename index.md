## 欢迎来到 My Wiki

这是一个基于 Jekyll 构建的轻量级 Wiki 知识库，具备以下特点：

- 📋 层级化侧边栏导航，支持多级菜单
- 📝 简洁的词条式排版，适合文档阅读
- 📱 自适应布局，兼容移动端和桌面端
- 🔍 面包屑导航，清晰显示当前位置
- ✨ 原生 Markdown 支持，语法高亮

## 快速开始

1. 在 `_config.yml` 中配置你的站点信息和侧边栏导航
2. 在根目录创建 `docs/` 目录，存放你的 Wiki 文档
3. 编写 Markdown 文档，指定 `layout: wiki` 即可使用该主题
4. 部署到 GitHub Pages 或其他静态站点托管平台
